A Vision for Bloomberg's Mobile Calendar

Bloomberg UX Internship Project 2023

For Bloomberg's 2023 UX internship, we designed a vision for the mobile calendar. This project investigated the current state of mobile productivity tools, and proposed an upgraded calendar that would integrated into Bloomberg's current system.

When: June - August 2023

Team: Mary Haws (me), Aniket Dwivedi

My Role: managing stakeholders, developing a research plan and protocol, conducting interviews, analyzing qualitative and quantitative data, presenting research findings, sketching, prototyping, conducting user tests, implementing feedback, pitching designs
